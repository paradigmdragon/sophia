- `PERSPECTIVE_MISSING`
- `RULE_CONFLICT`
- `BIT_INVALID`
- `FORMAT_MISMATCH`
- `NO_CAPABILITY`
- `INSUFFICIENT_EVIDENCE`

- `ADJUST_LOOP_GUARD`

- `CAPABILITY_MISMATCH`

Rule:
- Additive extension allowed.
- Existing code names must not be renamed or removed.
